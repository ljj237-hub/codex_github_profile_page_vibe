# GitHub Pages 개발자 포트폴리오

정적 HTML/CSS/JS 기반 포트폴리오 템플릿입니다.

## 파일 구조
- `index.html`: 메인 페이지
- `style.css`: 스타일
- `script.js`: 간단한 인터랙션(스크롤 reveal, 연도 자동 표시)

## 빠른 커스터마이징
`index.html`에서 아래 항목을 먼저 바꿔주세요.
- `YOUR NAME`
- `https://github.com/your-username`
- `you@example.com`
- 소개 문구, 프로젝트 항목

## GitHub Pages 배포
1. GitHub에 새 저장소를 생성합니다.
2. 현재 파일을 커밋/푸시합니다.
3. 저장소 `Settings` → `Pages`로 이동합니다.
4. `Build and deployment`에서 `Source`를 `Deploy from a branch`로 선택합니다.
5. `Branch`를 `main` / `/ (root)`로 선택 후 저장합니다.
6. 1~3분 뒤 `https://<github-username>.github.io/<repository-name>/`에서 확인합니다.

## 사용자/조직 사이트로 배포하려면
저장소 이름을 정확히 `your-username.github.io`로 만들면,
루트 도메인 `https://your-username.github.io/`로 바로 접속됩니다.
